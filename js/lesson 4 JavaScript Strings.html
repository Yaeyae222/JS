Lesson 4 JavaScript

Strings

What is String?

String ဆိုတာ စာသားတေရေးဖို့သုံးတာဖြစ်တယ်

HTML မာ စာရေးဖို့ paragraph သုံးရသလို JavaScript မာစာရေးဖို့ String သုံးရတယ်

Syntax Rule For String

String ရေးနည်း ( ၃ ) မျိုးရှိ

1. ' ' ( Single Quote )

2. " " ( Double Quote )

3. ` ` ( Template Lateral )

More Features Of String

1.String နှစ်ခုကို ပေါင်းလို့ရပါတယ် (Concatenation (Combine Strings Together) လို့ခေါ်တယ်)
နှစ်ခုထပ်လည်းပိုပေါင်းရတယ်

2. String နဲ့ Number ပေါင်းရင် Number က String အဖြစ်ပြောင်းသွားတယ် (Type Coersion or Automatic Type Conversion လို့ခေါ်တယ်)

Example

Single Quote String နဲ့ Double Quote String ရဲ့အားနည်းချက်များ

1. Single Quote String ထဲမာ Single Quote ပါမရ ( Double Quote String မာလည်းအတူတူပါပဲ )

2. အောက်ကိုတစ်ကြောင်းဆင်းမရ

Example

String ထဲမာသုံးလို့ရသော Characters များ

1 . Letters ( a , b , c )

2 . Number ( 1 , 2 , 3 )

3 . Symbol ( ! , @ , # )

4 . Escape Characters

Escape Characters

1. \' ( Single Quote String ထဲမာ Single Quote ထည့်ဖို့သုံး )

2. \" ( Double Quote String ထဲမာ Double Quote ထည့်ဖို့သုံး )

3. \n (အောက်တစ်ကြောင်းဆင်းဖို့သုံး ( HTML Line Break နဲ့တူ ) )

Template String အားသာချက်

1 . Interpolation ကြောင့်တန်ဖိုးတေတိုက်ရိုက်ထည့်ရ

2 . Multi Line String လို့ခေါ်တာကြောင့်အောက်တစ်ကြောင်းဆင်းရ

3 . Single Quote , Double Quote တေကို Escape Characters တေမလိုပဲသုံးရ

Example

What Should We Use?


ပုံသေအနေနဲ့ Single Quote သုံးပါ
